# Initiatives

Create one folder per **active** initiative under this directory.

**Naming:** `NNN-short-name` — three-digit zero-padded sequence (repo-wide, including `archive/`), hyphen, kebab-case topic. Existing folders use **001**–**018** from a one-time migration plus follow-on work (see [`archive/README.md`](archive/README.md)); the next id is **max existing + 1** (currently **022**). See `.agents/skills/housekeep/SKILL.md` for the allocation rule.

Shipped or abandoned handoffs belong in **`archive/`** (see [`archive/README.md`](archive/README.md)).

Inside each initiative folder, roles write handoffs using the shared protocol in [`../index.md`](../index.md).
